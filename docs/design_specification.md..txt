## Engineering Notes

1. Initial verification was performed using an ideal VCVS (E-element) model with gain = 11.
2. The ideal model correctly verified differential amplification and output centering around Vref = 1.65 V.
3. Because the E-element is an ideal dependent source, it does not model:
   - Dominant pole
   - Finite bandwidth
   - Input offset voltage
   - Input bias current
   - Noise
   - Real CMRR degradation
4. A full INA333 macromodel will be used in later simulations to evaluate realistic frequency response and non-ideal effects.
5. The design currently satisfies gain and signal-conditioning requirements for ECG acquisition.
