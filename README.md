# analog-front-end-ecg

Precision signal chain for biomedical sensing using
Texas Instruments INA333 instrumentation amplifier
and ADS1115 16-bit ADC.

## Project status
- [x] Day 1: LTspice installed, basic op-amp circuits simulated
- [ ] Week 1: Full signal chain simulation
- [ ] Week 2: PCB design in KiCad
- [ ] Week 3: PCB fabricated and tested
